DRAWER

The materialize-drawer component is used by the users to navigate throught the application.

USAGE

How to use

- Import the MaterializeDrawerModule in your module.

- Use the materialize-drawer tag in your template.

- Use the materialize-drawer-content tag into your materialize-drawer to display the content that the drawer will have.

- Use the materialize-drawer-container tag into your materialize-drawer to display the content that the rest of your main container will have.

THEMING

Materialize Angular supports a lot of SASS variables to change all the component style so that feel to modify component look and feel by changing the following SASS variables.

The Materialize Angular SASS variable names are created base on the following pattern:

"tagClassName" + (optional "-" + "modifierTagClassName" ) + "-" + "cssProperty"

.drawer-content

Name Default
$materialize-drawer-content-backgroundmap-get($theme-colors, 'white')
$materialize-drawer-content-width280px
$materialize-drawer-content-z-index$theme-zindex-2
$materialize-drawer-content-transition-function$theme-transition-function-in

.drawer-backdrop

Name Default
$materialize-drawer-backdrop-backgroundrgba(0, 0, 0, 0.5)
$materialize-drawer-backdrop-transition-function$theme-transition-function-out
$materialize-drawer-backdrop-transition-property$theme-property-opacity
$materialize-drawer-backdrop-z-index$theme-zindex-1
