USAGE

How to use

- Import the MaterializeNavbarModule in your module.

- Use the materialize-navbar tag in your template.

THEMING

Materialize Angular supports a lot of SASS variables to change all the component style so that feel to modify component look and feel by changing the following SASS variables.

The Materialize Angular SASS variable names are created base on the following pattern:

"tagClassName" + (optional "-" + "modifierTagClassName" ) + "-" + "cssProperty"

.navbar

Name Default
$materialize-navbar-background map-get($theme-colors, 'primary')
$materialize-navbar-color map-get($theme-colors, 'white')
$materialize-navbar-box-shadow $theme-shadow-sm
$materialize-navbar-font-family $theme-font-family-primary
$materialize-navbar-font-size $theme-font-size-lg
$materialize-navbar-font-weight $theme-font-weight-regular
$materialize-navbar-height 72px
$materialize-navbar-margin 0 0 $theme-space-sm
$materialize-navbar-padding 0 $theme-space-xs
