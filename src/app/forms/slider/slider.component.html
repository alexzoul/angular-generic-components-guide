SLIDER

The materialize-slider component is an easy way to see better the number input.

USAGE

How to use

- Import the MaterializeSliderModule in your module.

- Use the materialize-slider tag in your template.

- Use the materialize-slider-option tag into your materialize-slider to show and register each option.

- Use the materialize-slider-indicator tag in your materialize-slider to display the indicator you prefer.

API

Materialize Angular provides custom functions to add functionallity its components in different ways, the most common is using attributes in the component tag.

@Input Properties: Slider

Name Type Default Description
className string To specify an additional CSS class, use the "className" property. This applies to the component's first node.
disabled boolean false If true, the slider track and steps should be displayed in a disabled state.
required boolean false If true, the slider element will be required.
showLabels boolean true If true, option labels will be displayed.
showTicks boolean false If true, ticks on slider track will be displayed.
value number | string | boolean | null null The value of the slider element.

@Input Properties: Slider Option

Name Type Default Description
className string To specify an additional CSS class, use the "className" property. This applies to the component's first node.
value number | string | boolean | null null The value of the slider option element.

@Output Events: Slider

Name Signature Description
onChange onChangeEmitter(): EventEmitter<number | string | boolean | null> Event fired when the value is changed.

@Output Events: Slider Option

Name Signature Description
onClick onClickEmitter(): EventEmitter<number | string | boolean | null> Event fired when the option is clicked.
