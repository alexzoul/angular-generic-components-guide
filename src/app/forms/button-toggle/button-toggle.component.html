BUTTON TOGGLE

The materialize-button-toggle component is on/off toggles with the appearance of a button. The toggle can be configured as radio-buttons or checkboxes.

USAGE

How to use

- Import the MaterializeButtonToggleGroupModule in your module.

- Use the materialize-button-toggle-group tag in your template.

- Use the materialize-button-toggle tag into your materialize-button-toggle-group to show and register each button.

API

Materialize Angular provides custom functions to add functionallity its components in different ways, the most common is using attributes in the component tag.

@Input Properties: Button Toggle Group

Name Type Default Description
className string To specify an additional CSS class, use the "className" property. This applies to the component's first node.
disabled boolean false If true, all toggle buttons should be displayed in a disable state.
canUncheck boolean false If true, all options can be unchecked at the same time.
value string The id of the button toggle element.

@Input Properties: Button Toggle

Name Type Default Description
className string To specify an additional CSS class, use the "className" property. This applies to the component's first node.
disabled boolean false If true, the button should be displayed in a disable state.
value string The value of the button toggle

@Output Events: Button Toggle Group

Name Signature Description
onChange onChangeEmitter: EventEmitter<string> Event fired when the value is changed.

@Output Events: Button Toggle

Name Signature Description
onClick onClickEmitter: EventEmitter<string> Event fired when the button toggle is clicked.
