SELECT

The materialize-select component is the way to collect user information from a list of options.

USAGE

How to use

- Import the MaterializeSelectModule in your module.

- Use the materialize-select tag in your template.

- Use the materialize-select-option tag into your materialize-select.

The Materialize Angular SASS variable names are created base on the following pattern:

"tagClassName" + (optional "-" + "modifierTagClassName" ) + "-" + "cssProperty"
