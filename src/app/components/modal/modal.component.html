MODAL

The materialize-modal component is a dialog that can be open by calling the open method and it can be dismissed when touching outside dialog or by pressing the close button.

USAGE

How to use

- Import the MaterializeModalModule in your module.

- Use the materialize-modal tag in your template.

- To open the modal just call the modal open() method.

API

Materialize Angular provides custom functions to add functionallity its components in different ways, the most common is using attributes in the component tag.

@Input Properties

Name Type Default Description
className string To specify an additional CSS class, use the "className" property. This applies to the component's first node.
dismissOnBackdrop boolean true
hasBackdrop boolean true
hasCloseButton boolean true
isOpen boolean false
transitionDuration number 400

@Output Events

Name Signature Description
onOpen onOpenEmitter(): EventEmitter<void> "none"
onClose onCloseEmitter(): EventEmitter<string> "none"

THEMING

Materialize Angular supports a lot of SASS variables to change all the component style so that feel to modify component look and feel by changing the following SASS variables.

The Materialize Angular SASS variable names are created base on the following pattern:

"tagClassName" + (optional "-" + "modifierTagClassName" ) + "-" + "cssProperty"

Name Default
$materialize-modal-align-items center
$materialize-modal-justify-content center
