COLLAPSIBLE

The materialize-collapsible component you can control the state of a toolbar when the user interacts with it.

USAGE

How to use

- Import the MaterializeCollapsibleModule in your module.

- Use the materialize-collapsible tag in your template.

- Use the materialize-collapsible-title tag into your materialize-collapsible to display the collapsible title.

- Use the materialize-collapsible-content tag into your materialize-collapsible to display the collapsible content.

API

Materialize Angular provides custom functions to add functionallity its components in different ways, the most common is using attributes in the component tag.

@Input Properties

Name Type Default Description
className string To specify an additional CSS class, use the "className" property. This applies to the component's first node.
disabled boolean false If true, the collapsible title should not be enable to toggle content.
isOpen boolean false
showIndicator boolean true

@Output Events

Name Signature Description
onClose onCloseEmitter()EventEmitter<boolean> "none"
onOpen onOpenEmitter()EventEmitter<boolean> "none"

Materialize Angular supports a lot of SASS variables to change all the component style so that feel to modify component look and feel by changing the following SASS variables.

The Materialize Angular SASS variable names are created base on the following pattern:

"tagClassName" + (optional "-" + "modifierTagClassName" ) + "-" + "cssProperty"
