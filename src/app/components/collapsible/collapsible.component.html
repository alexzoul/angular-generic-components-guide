COLLAPSIBLE

The materialize-collapsible component you can control the state of a toolbar when the user interacts with it.

USAGE

How to use

- Import the MaterializeCollapsibleModule in your module.

- Use the materialize-collapsible tag in your template.

- Use the materialize-collapsible-title tag into your materialize-collapsible to display the collapsible title.

- Use the materialize-collapsible-content tag into your materialize-collapsible to display the collapsible content.
