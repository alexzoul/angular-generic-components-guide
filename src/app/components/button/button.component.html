BUTTON

The materialize-button replaces the standard html button. It's based in material design theme and a has multitude of options.

Can use different colors for text color and/or background being very modificable and flexible.

USAGE

How to use

- Import the MaterializeButtonModule in your module.

- Use the materialize-button tag in your template.

API

Materialize Angular provides custom functions to add functionallity its components in different ways, the most common is using attributes in the component tag.

@Input Properties

Name Type Default Description
className string " "
disabled boolean false
level text | accent | outline "accent"
rippleDuration number 250
type button | submit "button"

@Output Events

Name Signature Description
onClick onClickEmitter(): EventEmitter<void> "none"
onBlur onBlurEmitter(): EventEmitter<void> "none"

THEMING

Materialize Angular supports a lot of SASS variables to change all the component style so that feel to modify component look and feel by changing the following SASS variables.

The Materialize Angular SASS variable names are created base on the following pattern:

"tagClassName" + (optional "-" + "modifierTagClassName" ) + "-" + "cssProperty"
