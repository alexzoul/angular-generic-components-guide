RIPPLE

The materialize-ripple directive show how a user interact with the site. Can be applied to different components.

USAGE

How to use

- Import the MaterializeCommonModule in your module.

- Use the materializeRipple directive in your html tag.

API

Materialize Angular provides custom functions to add functionallity its components in different ways, the most common is using attributes in the component tag.

@Input Properties

Name Type Default Description
rippleDuration number 500
isRippleActive boolean true
isRippleCenter boolean false

THEMING

Materialize Angular supports a lot of SASS variables to change all the component style so that feel to modify component look and feel by changing the following SASS variables.

The Materialize Angular SASS variable names are created base on the following pattern:

"tagClassName" + (optional "-" + "modifierTagClassName" ) + "-" + "cssProperty"

Name Default
$materialize-ripple-backgorundrgba(map-get($theme-colors, 'black'), 0.1)
$materialize-ripple-border-radius$theme-radius-full
$materialize-ripple-transition-property$theme-property-opacity
$materialize-ripple-transition-function$theme-transition-function-in-out
