THEMING

Materialize Angular supports a lot of SASS variables to change all the component style so that feel to modify component look and feel by changing the following SASS variables.

The Materialize Angular SASS variable names are created base on the following pattern:

"tagClassName" + (optional "-" + "modifierTagClassName" ) + "-" + "cssProperty"

Name Default
$materialize-icon-color $materialize-color-gray-light
$materialize-icon-cursor pointer
$materialize-icon-display inline-block
$materialize-icon-outline none
$materialize-icon-text-transform lowercase
